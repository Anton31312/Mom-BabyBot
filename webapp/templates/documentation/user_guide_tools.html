Подробное руководство по инструментам

Счетчик схваток

Функциональность:

  • Запись времени начала и окончания каждой схватки
  • Расчет частоты и продолжительности схваток
  • Визуализация данных в виде графиков
  • История предыдущих сессий

Подробная инструкция:

  1. Перейдите в раздел "Инструменты" → "Счетчик схваток"
  2. Нажмите кнопку "Начать новую сессию"
  3. Когда начинается схватка, нажмите кнопку "Начало схватки"
  4. Когда схватка заканчивается, нажмите кнопку "Конец схватки"
  5. Повторяйте шаги 3-4 для каждой схватки
  6. Когда вы хотите завершить отслеживание, нажмите "Завершить сессию"

Приложение автоматически рассчитает и отобразит:

  • Среднюю продолжительность схваток
  • Средний интервал между схватками
  • График изменения частоты схваток

Когда обращаться к врачу: Если схватки происходят регулярно с интервалом 5-7 минут и длятся около 1 минуты каждая в течение часа, это может указывать на начало активной фазы родов. Также немедленно обратитесь к врачу, если отошли воды или появились кровянистые выделения.

Счетчик шевелений

Функциональность:

  • Запись времени каждого шевеления
  • Расчет времени, необходимого для 10 шевелений
  • История предыдущих сессий
  • Визуализация активности ребенка по дням

Подробная инструкция:

  1. Перейдите в раздел "Инструменты" → "Счетчик шевелений"
  2. Нажмите кнопку "Начать новую сессию"
  3. Каждый раз, когда вы чувствуете шевеление ребенка, нажимайте кнопку "Записать шевеление"
  4. После 10 шевелений или когда вы хотите завершить сессию, нажмите "Завершить"

Приложение автоматически рассчитает и отобразит:

  • Время, за которое произошло 10 шевелений
  • Общее количество шевелений за сессию
  • График активности по дням

Рекомендации по использованию: Лучше всего отслеживать шевеления в одно и то же время каждый день, когда ребенок обычно активен (например, после еды). Рекомендуется проводить подсчет шевелений ежедневно, начиная с 28 недели беременности.

Когда обращаться к врачу: Если вы заметили значительное снижение активности ребенка (например, если для 10 шевелений требуется более 2 часов) или если шевеления прекратились, немедленно обратитесь к врачу.

Таймер сна

Функциональность:

  • Отслеживание времени сна ребенка
  • Разделение на дневной и ночной сон
  • Статистика и графики сна
  • Работа в фоновом режиме

Подробная инструкция:

  1. Перейдите в раздел "Инструменты" → "Таймер сна"
  2. Выберите профиль ребенка (если у вас несколько детей)
  3. Выберите тип сна (дневной или ночной)
  4. Нажмите кнопку "Начать" когда ребенок засыпает
  5. Нажмите кнопку "Завершить" когда ребенок просыпается

Приложение автоматически рассчитает и отобразит:

  • Общую продолжительность сна
  • Соотношение дневного и ночного сна
  • График сна по дням
  • Сравнение с рекомендуемыми нормами для возраста

Особенности: Таймер сна продолжает работать даже если приложение свернуто или закрыто. Данные синхронизируются при восстановлении соединения.

Отслеживание кормления

Функциональность:

  • Отслеживание грудного вскармливания (время, продолжительность, грудь)
  • Отслеживание кормления из бутылочки (время, количество, тип смеси)
  • Статистика и графики кормления
  • Напоминания о следующем кормлении

Подробная инструкция для грудного вскармливания:

  1. Перейдите в раздел "Инструменты" → "Отслеживание кормления"
  2. Выберите профиль ребенка (если у вас несколько детей)
  3. Выберите тип кормления "Грудное вскармливание"
  4. Выберите грудь (левая, правая или обе)
  5. Нажмите "Начать" в начале кормления
  6. Нажмите "Завершить" по окончании кормления

Приложение автоматически предложит использовать другую грудь при следующем кормлении для поддержания баланса.

Подробная инструкция для кормления из бутылочки:

  1. Перейдите в раздел "Инструменты" → "Отслеживание кормления"
  2. Выберите профиль ребенка (если у вас несколько детей)
  3. Выберите тип кормления "Из бутылочки"
  4. Введите количество молока/смеси в мл
  5. При необходимости выберите тип смеси из списка или добавьте новый
  6. Нажмите "Сохранить"

Календарь прививок

Функциональность:

  • Отображение рекомендуемых прививок по возрасту
  • Разделение на обязательные и рекомендуемые прививки
  • Отметка о сделанных прививках
  • Напоминания о предстоящих прививках
  • Информация о каждой вакцине

Подробная инструкция:

  1. Перейдите в раздел "Инструменты" → "Календарь прививок"
  2. Выберите профиль ребенка (если у вас несколько детей)
  3. Просмотрите список рекомендуемых прививок, отсортированных по возрасту
  4. Нажмите на прививку, чтобы увидеть подробную информацию о ней
  5. Чтобы отметить прививку как сделанную, нажмите кнопку "Отметить как выполненную"
  6. Введите дату вакцинации и при необходимости добавьте комментарий
  7. Нажмите "Сохранить"

Вы можете фильтровать прививки по категориям (обязательные/рекомендуемые) или по статусу (сделанные/предстоящие).

Важно: Календарь прививок составлен в соответствии с рекомендациями ВОЗ и Министерства здравоохранения России, но всегда консультируйтесь с педиатром перед вакцинацией.
