Общая информация

О приложении

Mom&BabyBot — это комплексное приложение для будущих и новых родителей, которое помогает отслеживать беременность, развитие ребенка и предоставляет полезные инструменты для ухода за малышом.

Наше приложение доступно как в виде веб-интерфейса, так и в виде Telegram бота, что позволяет вам использовать его на любом устройстве и в любое время.

Навигация по приложению

Приложение разделено на несколько основных разделов:

Информационные разделы

  • Беременность — информация о течении беременности по триместрам
  • Развитие ребенка — информация о развитии ребенка по возрастам
  • Питание — рекомендации по питанию для беременных, кормящих мам и детей

Инструменты

  • Счетчик схваток — для отслеживания схваток во время родов
  • Счетчик шевелений — для отслеживания активности ребенка во время беременности
  • Таймер сна — для отслеживания режима сна ребенка
  • Отслеживание кормления — для записи грудного вскармливания и кормления из бутылочки
  • Профили детей — для управления данными о детях и их развитии
  • Календарь прививок — для отслеживания графика вакцинации

Начало работы

Чтобы начать использовать приложение:

  1. Создайте учетную запись или войдите через Telegram
  2. Заполните свой профиль, указав информацию о беременности или ребенке
  3. Исследуйте информационные разделы или начните использовать инструменты

Все данные автоматически синхронизируются между веб-приложением и Telegram ботом, если вы используете одну учетную запись.

Интерфейс приложения

Наше приложение использует современный неоморфный и глазоморфный дизайн с теплыми, пастельными цветами, создающими комфортную атмосферу.

Неоморфные элементы

Элементы с мягкими тенями и выпуклым эффектом, которые можно нажимать и с которыми можно взаимодействовать.

Глазоморфные элементы

Полупрозрачные элементы с эффектом матового стекла, которые используются для выделения информации.
