Руководство по функциям для беременных

Информационный раздел "Беременность"

В этом разделе вы найдете подробную информацию о течении беременности, разделенную по триместрам:

Первый триместр (1-13 недели)

Информация о ранних стадиях беременности, включая:

  • Признаки беременности
  • Развитие плода по неделям
  • Рекомендации по питанию и образу жизни
  • Важные медицинские обследования

Второй триместр (14-27 недели)

Информация о среднем периоде беременности, включая:

  • Развитие плода по неделям
  • Физические изменения в организме матери
  • Рекомендации по физической активности
  • Подготовка к родам

Третий триместр (28-40 недели)

Информация о последнем периоде беременности, включая:

  • Развитие плода по неделям
  • Подготовка к родам
  • Признаки начала родов
  • Сбор сумки в роддом

Счетчик схваток

Счетчик схваток помогает отслеживать частоту, продолжительность и интенсивность схваток во время родов.

Как использовать счетчик схваток:

  1. Перейдите в раздел "Инструменты" → "Счетчик схваток"
  2. Нажмите кнопку "Начать" для запуска новой сессии
  3. Нажимайте кнопку "Записать схватку" каждый раз, когда начинается схватка
  4. Нажмите кнопку "Завершить" для окончания сессии

Приложение автоматически рассчитает частоту и продолжительность схваток, а также покажет график их интенсивности.

Совет: Обычно рекомендуется обращаться в роддом, когда схватки происходят с интервалом 5-7 минут и длятся около 1 минуты каждая.

Счетчик шевелений

Счетчик шевелений помогает отслеживать активность ребенка во время беременности, что является важным показателем его здоровья.

Как использовать счетчик шевелений:

  1. Перейдите в раздел "Инструменты" → "Счетчик шевелений"
  2. Нажмите кнопку "Начать" для запуска новой сессии
  3. Нажимайте кнопку "Записать шевеление" каждый раз, когда чувствуете движение ребенка
  4. Нажмите кнопку "Завершить" для окончания сессии

Приложение сохранит данные о шевелениях и покажет статистику активности ребенка.

Важно: Врачи рекомендуют отслеживать время, за которое вы насчитаете 10 шевелений. В норме это должно занимать не более 2 часов. Если вы заметили значительное снижение активности ребенка, обратитесь к врачу.

Раздел "Питание для беременных"

В этом разделе вы найдете рекомендации по питанию во время беременности:

  • Необходимые питательные вещества по триместрам
  • Продукты, которые следует избегать
  • Примеры сбалансированного меню
  • Рекомендации по приему витаминов и добавок

Совет: Всегда консультируйтесь с вашим врачом перед внесением значительных изменений в рацион или началом приема добавок.
