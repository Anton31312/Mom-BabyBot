Синхронизация данных

Синхронизация с Telegram ботом

Mom&BabyBot доступен как в виде веб-приложения, так и в виде Telegram бота. Все данные автоматически синхронизируются между ними, что позволяет вам использовать наиболее удобный интерфейс в зависимости от ситуации.

Как связать веб-приложение с Telegram ботом:

  1. Найдите бота @MomBabyBot в Telegram и начните диалог
  2. Нажмите кнопку "Начать" или отправьте команду /start
  3. Следуйте инструкциям бота для создания учетной записи
  4. В веб-приложении нажмите "Войти через Telegram"
  5. Введите код, который вы получите от бота

После связывания все данные будут автоматически синхронизироваться между веб-приложением и ботом.

Работа в офлайн-режиме

Веб-приложение Mom&BabyBot может работать даже без подключения к интернету. Все данные сохраняются локально и синхронизируются с сервером при восстановлении соединения.

Как работает офлайн-режим:

  • Все данные кэшируются в локальном хранилище браузера
  • Таймеры и счетчики продолжают работать даже без подключения к интернету
  • При восстановлении соединения данные автоматически синхронизируются с сервером
  • В случае конфликта данных приложение предложит выбрать, какие данные сохранить

Совет: Для наилучшей работы в офлайн-режиме рекомендуется использовать современные браузеры, такие как Chrome, Firefox или Safari.

Управление данными

Экспорт данных:

  1. Перейдите в раздел "Настройки" → "Управление данными"
  2. Нажмите кнопку "Экспортировать данные"
  3. Выберите формат экспорта (JSON или CSV)
  4. Выберите категории данных для экспорта
  5. Нажмите "Экспортировать"

Файл с данными будет загружен на ваше устройство.

Удаление данных:

  1. Перейдите в раздел "Настройки" → "Управление данными"
  2. Нажмите кнопку "Удалить данные"
  3. Выберите категории данных для удаления
  4. Введите "УДАЛИТЬ" для подтверждения
  5. Нажмите "Удалить"

Внимание: Удаление данных необратимо. Рекомендуется сначала экспортировать данные.

Уведомления

Mom&BabyBot может отправлять уведомления о важных событиях как в веб-приложении, так и через Telegram бота.

Настройка уведомлений:

  1. Перейдите в раздел "Настройки" → "Уведомления"
  2. Выберите типы уведомлений, которые вы хотите получать
  3. Выберите способы получения уведомлений (веб, Telegram, или оба)
  4. Нажмите "Сохранить"

Для получения уведомлений в веб-приложении необходимо разрешить уведомления в браузере.

Типы уведомлений:

  • Напоминания о прививках — уведомления о предстоящих прививках
  • Активные таймеры — уведомления о работающих таймерах сна или кормления
  • Статистика — еженедельные отчеты о сне, кормлении и развитии ребенка
  • Советы — полезные советы по уходу за ребенком в соответствии с его возрастом
